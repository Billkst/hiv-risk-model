# Privacy Policy

## Data Collection

This plugin does NOT collect or store any personal data.

## Data Processing

- All predictions are performed locally within the plugin
- No data is sent to external servers
- Medical data is processed in memory only
- No persistent storage of user data

## Model

- The AI model is packaged within the plugin
- Model predictions are performed locally
- No external API calls for predictions

## Data Usage

Input data (medical statistics) is used solely for:
- Risk level prediction
- Feature contribution analysis
- Generating assessment reports

## Data Retention

- No data is retained after processing
- All computations are stateless
- No logs of medical data are kept
